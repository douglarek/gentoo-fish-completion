# gentoo-fish-completion

This project focuses on providing Fish completion support to various Gentoo tools that lack completion scripts upstream.

* [gentoo-zsh-completion](https://github.com/gentoo/zsh-completion)
* [eix](https://github.com/vaeth/eix)
* [pkgdev](https://github.com/pkgcore/pkgdev)
* [pkgcheck](https://github.com/pkgcore/pkgcheck/)
